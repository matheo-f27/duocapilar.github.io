¿Te ínteresa saber como hicimos nuestros productos?

la barra fijadora y el tónico capilar se basa en la utilizacion de los ingredientes naturales, reconocidos por los beneficios que contienen para el cabello. Estos productos son aptos para su uso en toda la población, al estar compuestos por exclusivamente productos naturales que permiten ampliar su alcance a un mayor número de personas.

Para la elaboracion de los productos capilares, se realiza una selección de materias primas y diferentes aceites de productos naturales. obtenidos de distintos puntos de la región; los cuales contienen beneficios que contribuyen a la salud del cabello

Barra fijadora

para la barra fijadora se dispone y prepara un material adecuado de laboratorio, se asegura la exactitud para cada medición y se usan procesos industriales: como lo son la destilación, de igual manera se usa un proceso de extraccion termica para obtener cada uno de los aceites utilizados. se utiliza un baño maria para la integracion de los componentes para luego solidificar y quedar asi una barra.

Tonico capilar

para la elaboracion de nuestro tonico capilar.se dispone y se prepara el material adecuado de laboratorio, para este utilizamos procesos industriales como lo es la centifruga en la cual se realiza una extraccion liquido solido para obtener la vitamina a, se corta, masera y licúa los cristales de la sabila para obtener el gel, se destila el romero para extraer el hidrolato de este; se unifican todos los componentes para la elaboracion del tonico.
